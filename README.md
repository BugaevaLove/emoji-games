# 🎰 Emoji Games Bot

Асинхронный Telegram‑бот с играми на эмодзи, VIP‑системой, рефералами и админ‑панелью.

## 🚀 Быстрый старт
1. `pip install -r requirements.txt`
2. Создайте `.env` с `BOT_TOKEN` и `ADMIN_IDS`
3. `python bot.py`

## 🎮 Игры
- 🎲 Кубик, 🎯 Дартс, 🎳 Боулинг, ⚽ Футбол, 🏀 Баскетбол, 🎰 Слоты
- Два режима: обычный (RTP 95%) и Telegram

## 👑 VIP
12 уровней, награды, кэшбек до 15%, рейкбек до 12%

## 👥 Рефералы
10% от депозитов друга + 12% от его оборота

## 📊 Админка
`/admin` – статистика, поиск, выдача/списание, бан, рассылка
